The theorem

Matthew Litwin 2026-05-18

Consider a smooth one-parameter family of transformations of paths:

q(t)↦qϵ(t),q0(t)=q(t),q(t) \mapsto q_\epsilon(t), \qquad q_0(t) = q(t), q(t)↦qϵ​(t),q0​(t)=q(t),

δqi:=∂qϵi∂ϵ∣ϵ=0.\delta q^i := \frac{\partial q_\epsilon^i}{\partial \epsilon}\bigg|_{\epsilon = 0}. δqi:=∂ϵ∂qϵi​​​ϵ=0​.

The transformation is a symmetry of the action if for every smooth path qqq,

ddϵS[qϵ]∣ϵ=0=0\frac{d}{d\epsilon} S[q_\epsilon]\bigg|_{\epsilon=0} = 0 dϵd​S[qϵ​]​ϵ=0​=0

— the variation δS\delta SδS vanishes for all paths, not only solutions.

Noether’s theorem. For each such symmetry, the quantity

Q:=∂L∂q˙i δqi=pi δqi\boxed{\quad Q := \frac{\partial L}{\partial \dot q^i}\, \delta q^i = p_i\, \delta q^i \quad} Q:=∂q˙​i∂L​δqi=pi​δqi​

is conserved along solutions of the Euler–Lagrange equations: dQ/dt=0dQ/dt = 0dQ/dt=0.

Quasi-symmetries (action shifts by a total derivative). If the variation produces

δL=dFdt\delta L = \frac{dF}{dt} δL=dtdF​

for some function F(q,t)F(q, t)F(q,t) — rather than vanishing — the symmetry is called a quasi-symmetry (or “Noether symmetry up to a total derivative”). The conserved quantity is

Q=∂L∂q˙i δqi−F.Q = \frac{\partial L}{\partial \dot q^i}\, \delta q^i - F. Q=∂q˙​i∂L​δqi−F.

The FFF term picks up the would-be boundary contribution from δL\delta LδL.
