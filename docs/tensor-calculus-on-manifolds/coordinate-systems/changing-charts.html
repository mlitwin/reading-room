Changing charts

Matthew Litwin 2026-06-23

The two charts on S2S^2S2 describe the same surface. A vector or a function at a point has a single intrinsic identity; what changes between charts is the components. This page sets up the bookkeeping.

The Jacobian

Between the two charts the transition is

θ~=θ,φ~=φ+αcos⁡θ,\tilde\theta = \theta, \qquad \tilde\varphi = \varphi + \alpha \cos\theta, θ~=θ,φ~​=φ+αcosθ,

with inverse

θ=θ~,φ=φ~−αcos⁡θ~.\theta = \tilde\theta, \qquad \varphi = \tilde\varphi - \alpha\cos\tilde\theta. θ=θ~,φ=φ~​−αcosθ~.

The Jacobians are

J=∂(θ~,φ~)∂(θ,φ)=(10−αsin⁡θ1),J−1=∂(θ,φ)∂(θ~,φ~)=(10αsin⁡θ~1).J = \frac{\partial(\tilde\theta, \tilde\varphi)}{\partial(\theta, \varphi)} = \begin{pmatrix} 1 & 0 \\ -\alpha\sin\theta & 1 \end{pmatrix}, \qquad J^{-1} = \frac{\partial(\theta, \varphi)}{\partial(\tilde\theta, \tilde\varphi)} = \begin{pmatrix} 1 & 0 \\ \alpha\sin\tilde\theta & 1 \end{pmatrix}. J=∂(θ,φ)∂(θ~,φ~​)​=(1−αsinθ​01​),J−1=∂(θ~,φ~​)∂(θ,φ)​=(1αsinθ~​01​).

Basis vectors transform with J−1J^{-1}J−1

A coordinate basis vector ∂μ~\partial_{\tilde\mu}∂μ~​​ at a point is the derivation ∂/∂x~μ\partial/\partial \tilde{x}^\mu∂/∂x~μ. By the chain rule applied to any smooth function fff:

∂θ~f=∂θ∂θ~ ∂θf+∂φ∂θ~ ∂φf=∂θf+αsin⁡θ~ ∂φf.\partial_{\tilde\theta} f = \frac{\partial \theta}{\partial \tilde\theta}\, \partial_\theta f + \frac{\partial \varphi}{\partial \tilde\theta}\, \partial_\varphi f = \partial_\theta f + \alpha\sin\tilde\theta\, \partial_\varphi f. ∂θ~​f=∂θ~∂θ​∂θ​f+∂θ~∂φ​∂φ​f=∂θ​f+αsinθ~∂φ​f.

So

∂θ~=∂θ+αsin⁡θ~ ∂φ,∂φ~=∂φ,\partial_{\tilde\theta} = \partial_\theta + \alpha\sin\tilde\theta\, \partial_\varphi, \qquad \partial_{\tilde\varphi} = \partial_\varphi, ∂θ~​=∂θ​+αsinθ~∂φ​,∂φ~​​=∂φ​,

matching what the previous page derived by differentiating the embedded parametrization.

In matrix form, with column vectors of basis elements:

(∂θ~∂φ~)=J−1(∂θ∂φ).\begin{pmatrix} \partial_{\tilde\theta} \\ \partial_{\tilde\varphi} \end{pmatrix} = J^{-1} \begin{pmatrix} \partial_\theta \\ \partial_\varphi \end{pmatrix}. (∂θ~​∂φ~​​​)=J−1(∂θ​∂φ​​).

The basis transforms with the inverse Jacobian. This is the defining property of a covariant index — moving with the basis, opposite to how vector components move.

Components transform with JJJ

A tangent vector vvv has components vμv^\muvμ in the standard chart and v~μ~\tilde v^{\tilde\mu}v~μ~​ in the skew chart. The intrinsic identity

v=vθ ∂θ+vφ ∂φ=v~θ~ ∂θ~+v~φ~ ∂φ~v = v^\theta\, \partial_\theta + v^\varphi\, \partial_\varphi = \tilde v^{\tilde\theta}\, \partial_{\tilde\theta} + \tilde v^{\tilde\varphi}\, \partial_{\tilde\varphi} v=vθ∂θ​+vφ∂φ​=v~θ~∂θ~​+v~φ~​∂φ~​​

combined with the basis transformation above gives

v~θ~=vθ,v~φ~=vφ−αsin⁡θ vθ,\tilde v^{\tilde\theta} = v^\theta, \qquad \tilde v^{\tilde\varphi} = v^\varphi - \alpha\sin\theta\, v^\theta, v~θ~=vθ,v~φ~​=vφ−αsinθvθ,

or in matrix form v~=Jv\tilde v = J vv~=Jv. Vector components transform with JJJ, opposite to the basis — this is the contravariant transformation rule, and the reason vector indices are written up.

A covector ω\omegaω pairs against vvv to give a chart-independent number ω(v)=ωμvμ\omega(v) = \omega_\mu v^\muω(v)=ωμ​vμ. For that pairing to be invariant under chart change, the covector components must transform with J−1J^{-1}J−1:

ω~θ~=ωθ+αsin⁡θ ωφ,ω~φ~=ωφ.\tilde \omega_{\tilde\theta} = \omega_\theta + \alpha\sin\theta\, \omega_\varphi, \qquad \tilde \omega_{\tilde\varphi} = \omega_\varphi. ω~θ~​=ωθ​+αsinθωφ​,ω~φ~​​=ωφ​.

This is the covariant rule — index down.

A worked pairing

Take the function f(p)=Z(p)=cos⁡θ(p)f(p) = Z(p) = \cos\theta(p)f(p)=Z(p)=cosθ(p) (the embedded ZZZ-coordinate). Its differential is the covector dfdfdf, with standard-chart components

Same components — because θ~=θ\tilde\theta = \thetaθ~=θ and fff depends only on θ\thetaθ. Sanity check.

Now take the vector v=∂φv = \partial_\varphiv=∂φ​ (rotation around the ZZZ-axis). Its standard components are (vθ,vφ)=(0,1)(v^\theta, v^\varphi) = (0, 1)(vθ,vφ)=(0,1). Skew components:

Identical again, because ∂φ=∂φ~\partial_\varphi = \partial_{\tilde\varphi}∂φ​=∂φ~​​ in our shear.

The components changed; the number didn’t.
