This is the very first piece in the Reading Room. If you're seeing this rendered, the build pipeline is alive.

The site you're reading turns markdown files in `content/` into:

- A landing page that lists every piece.
- A reader page per piece (this one).
- An `index.json` the iOS app pulls from.

Nothing fancy here — just running text to confirm typography looks right. A second paragraph for good measure: the quick brown fox jumps over the lazy dog, and then keeps going for long enough that we can see how line height, justification, and the maximum line length actually feel in real prose rather than in a one-line filler.
