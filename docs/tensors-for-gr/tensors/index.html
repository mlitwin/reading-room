Tensors

Matthew Litwin 2026-05-21

A tensor is a multilinear gadget that eats some vectors and some covectors and returns a real number. The same object has two names: physicists call it an (r,s)(r, s)(r,s)-tensor with rrr upper and sss lower indices; mathematicians call it an element of TpM⊗r⊗Tp∗M⊗sT_p M^{\otimes r} \otimes T^*_p M^{\otimes s}Tp​M⊗r⊗Tp∗​M⊗s. They are identical.

Four pages: the multilinear-map definition and tensor product; how indices and the transformation rule encode it; the side-by-side dictionary between index notation, coordinate-free notation, and (anti-symmetric) kkk-forms; tensors and forms on S2S^2S2 worked out in components.
