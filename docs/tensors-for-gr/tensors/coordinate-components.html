Coordinate components and the transformation rule

Matthew Litwin 2026-05-21

The physicist’s working definition of a tensor is exactly the multilinear-map definition expressed in components.

The transformation rule

A choice of chart xμx^\muxμ around ppp gives a coordinate basis for Tsr(TpM)T^r_s(T_p M)Tsr​(Tp​M) and so a component array Tμ1⋯μrν1⋯νsT^{\mu_1 \cdots \mu_r}{}_{\nu_1 \cdots \nu_s}Tμ1​⋯μr​ν1​⋯νs​​. Under xμ↦x′μ′(x)x^\mu \mapsto x'^{\mu'}(x)xμ↦x′μ′(x), the components transform as

T′μ1′⋯μr′ν1′⋯νs′=∂x′μ1′∂xμ1⋯∂x′μr′∂xμr  ∂xν1∂x′ν1′⋯∂xνs∂x′νs′  Tμ1⋯μrν1⋯νs.T'^{\mu'_1 \cdots \mu'_r}{}_{\nu'_1 \cdots \nu'_s} = \frac{\partial x'^{\mu'_1}}{\partial x^{\mu_1}} \cdots \frac{\partial x'^{\mu'_r}}{\partial x^{\mu_r}}\; \frac{\partial x^{\nu_1}}{\partial x'^{\nu'_1}} \cdots \frac{\partial x^{\nu_s}}{\partial x'^{\nu'_s}}\; T^{\mu_1 \cdots \mu_r}{}_{\nu_1 \cdots \nu_s}. T′μ1′​⋯μr′​ν1′​⋯νs′​​=∂xμ1​∂x′μ1′​​⋯∂xμr​∂x′μr′​​∂x′ν1′​∂xν1​​⋯∂x′νs′​∂xνs​​Tμ1​⋯μr​ν1​⋯νs​​.

Every upper index gets one factor of ∂x′/∂x\partial x'/\partial x∂x′/∂x; every lower index gets one factor of ∂x/∂x′\partial x/\partial x'∂x/∂x′. This is the tensor transformation law.

A component array satisfying this rule on every chart overlap defines a tensor — and conversely, the rule is exactly what’s needed for the abstract multilinear map to be coordinate-independent.

Einstein summation

A repeated index appearing once up and once down in the same monomial is summed from 111 to nnn:

vμωμ:=∑μ=1nvμωμ,Tμνvν:=∑ν=1nTμνvν.v^\mu \omega_\mu := \sum_{\mu=1}^{n} v^\mu \omega_\mu, \qquad T^\mu{}_\nu v^\nu := \sum_{\nu=1}^{n} T^\mu{}_\nu v^\nu. vμωμ​:=μ=1∑n​vμωμ​,Tμν​vν:=ν=1∑n​Tμν​vν.

Indices in the same position (both up, or both down) are not summed and usually indicate an error in the calculation. The convention is so consistent that the summation symbol can be omitted everywhere; the index positions enforce the rule.

A free index appears once on each side of an equation, at the same vertical position; equations are read as holding for every value of every free index. A dummy (or summed) index appears as an up-down pair on the same side and can be renamed at will.

Operations in component form

The operations on tensors all have one-line component expressions.

  • Tensor product: (S⊗T)μ1⋯μr1+r2ν1⋯νs1+s2=Sμ1⋯μr1ν1⋯νs1  Tμr1+1⋯νs1+1⋯(S \otimes T)^{\mu_1 \cdots \mu_{r_1+r_2}}{}_{\nu_1 \cdots \nu_{s_1+s_2}} = S^{\mu_1 \cdots \mu_{r_1}}{}_{\nu_1 \cdots \nu_{s_1}}\; T^{\mu_{r_1+1} \cdots}{}_{\nu_{s_1+1} \cdots}(S⊗T)μ1​⋯μr1​+r2​​ν1​⋯νs1​+s2​​​=Sμ1​⋯μr1​​ν1​⋯νs1​​​Tμr1​+1​⋯νs1​+1​⋯​.
  • Contraction of the kkk-th upper with the ℓ\ellℓ-th lower index: set them equal and sum.
  • Symmetrization: T(μν):=12(Tμν+Tνμ)T_{(\mu\nu)} := \tfrac{1}{2}(T_{\mu\nu} + T_{\nu\mu})T(μν)​:=21​(Tμν​+Tνμ​); for kkk indices, average over all permutations.
  • Antisymmetrization: T[μν]:=12(Tμν−Tνμ)T_{[\mu\nu]} := \tfrac{1}{2}(T_{\mu\nu} - T_{\nu\mu})T[μν]​:=21​(Tμν​−Tνμ​); for kkk indices, sign-weighted average.

The “transforms like a tensor” diagnostic

Not every indexed quantity is a tensor. The Christoffel symbols of the connection section, for example, are not tensors — they pick up an inhomogeneous piece in their transformation law:

Γ′ρ′μ′ν′=∂x′ρ′∂xρ∂xμ∂x′μ′∂xν∂x′ν′ Γρμν  +  ∂x′ρ′∂xρ∂2xρ∂x′μ′∂x′ν′.\Gamma'^{\rho'}{}_{\mu'\nu'} = \frac{\partial x'^{\rho'}}{\partial x^\rho} \frac{\partial x^\mu}{\partial x'^{\mu'}} \frac{\partial x^\nu}{\partial x'^{\nu'}}\, \Gamma^{\rho}{}_{\mu\nu} \;+\; \frac{\partial x'^{\rho'}}{\partial x^\rho} \frac{\partial^2 x^\rho}{\partial x'^{\mu'} \partial x'^{\nu'}}. Γ′ρ′μ′ν′​=∂xρ∂x′ρ′​∂x′μ′∂xμ​∂x′ν′∂xν​Γρμν​+∂xρ∂x′ρ′​∂x′μ′∂x′ν′∂2xρ​.

The second-derivative term spoils tensoriality. The presence or absence of such a term, in any candidate construction, is the diagnostic: if the array transforms multiplicatively as above, it’s a tensor; otherwise it isn’t.

The equivalent abstract diagnostic is C∞(M)C^\infty(M)C∞(M)-linearity in every slot. A multilinear map on vector fields and 1-forms that is R\mathbb{R}R-linear and C∞(M)C^\infty(M)C∞(M)-linear (not merely R\mathbb{R}R-linear) comes from a tensor field. Multilinearity over the smooth functions is the operational reason the value at a point depends only on values at that point.
