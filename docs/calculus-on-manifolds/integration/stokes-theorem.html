Stokes's theorem

Matthew Litwin 2026-05-18

A manifold with boundary is a topological space modeled locally on Rn\mathbb{R}^nRn (interior points) or on Hn:={x∈Rn:xn≥0}\mathbb{H}^n := \{x \in \mathbb{R}^n : x^n \geq 0\}Hn:={x∈Rn:xn≥0} (boundary points). The boundary ∂M\partial M∂M is the set of points mapped to {xn=0}\{x^n = 0\}{xn=0} by some (hence every) chart; it’s itself a smooth (n−1)(n-1)(n−1)-manifold (without boundary).

Boundary orientation. If MMM is oriented with form Ω\OmegaΩ and ν\nuν is an outward-pointing vector field along ∂M\partial M∂M, the induced orientation on ∂M\partial M∂M is represented by ινΩ∣∂M\iota_\nu \Omega|_{\partial M}ιν​Ω∣∂M​. (“Outward normal first” convention.)

Stokes’s theorem. Let MMM be a smooth oriented nnn-manifold with boundary, and let ω∈Ωn−1(M)\omega \in \Omega^{n-1}(M)ω∈Ωn−1(M) have compact support. Then

∫Mdω  =  ∫∂Mω\boxed{\quad \int_M d\omega \;=\; \int_{\partial M} \omega \quad} ∫M​dω=∫∂M​ω​

with ∂M\partial M∂M given the induced orientation. (If MMM has no boundary, both sides vanish when ω\omegaω has compact support.)
