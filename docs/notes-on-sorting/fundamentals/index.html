Fundamentals

Matthew Litwin 2026-05-17

Two things matter when comparing sorting algorithms. The first is complexity — how the running time grows with the number of elements. The second is stability — whether elements that compare equal keep their original relative order. Both are easy to get vague about, so it’s worth being precise.

The pages that follow take each in turn before moving on to the algorithms themselves.
