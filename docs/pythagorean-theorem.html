A Note on the Pythagorean Theorem

Matthew Litwin 2026-05-14 mathgeometry

For a right triangle with legs aaa and bbb and hypotenuse ccc:

a2+b2=c2a^2 + b^2 = c^2 a2+b2=c2

Rearranging:

c=a2+b2c = \sqrt{a^2 + b^2} c=a2+b2​

A worked example with a=3a = 3a=3, b=4b = 4b=4:

c=32+42=25=5c = \sqrt{3^2 + 4^2} = \sqrt{25} = 5 c=32+42​=25​=5

There is something pleasingly tidy about this case — three integers, no leftover. Such triples (3,4,53, 4, 53,4,5) are called Pythagorean triples, and they have a parametric form: for any integers m>n>0m > n > 0m>n>0,

a=m2−n2,b=2mn,c=m2+n2a = m^2 - n^2, \quad b = 2mn, \quad c = m^2 + n^2 a=m2−n2,b=2mn,c=m2+n2
