Paradigm grid — working demonstration

Self-contained snapshot of the layout pattern as shipped (deduco, active voice). CSS is copied verbatim from site/reader/css/06-paradigm.css; the DOM matches cards.js. See paradigm-grid-pattern.md.

Drag the slider (or type a width) to change the container width and watch the columns wrap into bands — rows stay aligned within a band via CSS subgrid, and the row-header badge gutter rebuilds per band.
